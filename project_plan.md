# 줄눈시공 전문 홈페이지

## 1. 프로젝트 설명
인테리어 줄눈시공 전문 업체를 위한 랜딩 페이지. 포트폴리오 갤러리, 고객 후기, FAQ, 무료 견적 신청 폼을 제공해 신뢰도를 높이고 잠재 고객의 문의를 유도하는 단일 페이지 웹사이트.

**타겟 유저**: 인테리어 시공을 고려 중인 주거/상업 공간 소유자
**핵심 가치**: 전문성과 신뢰성을 시각적으로 강조, 간단한 견적 신청으로 전환율 향상

## 2. 페이지 구조
- `/` - 메인 홈 (단일 페이지, 스크롤 내비게이션)

### 섹션 구성
- 네비게이션 바 (고정)
- 메인 배너 (동영상 히어로)
- 포트폴리오 & 고객 후기
- 작업 범위 & 차별점
- SNS 연동 (인스타그램 피드)
- FAQ
- 견적 신청 폼
- 푸터
